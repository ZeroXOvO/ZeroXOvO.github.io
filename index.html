My Projects

API Project

A car park booking API that allows a user to reserve a parking space for a car parking space, it also allows the user to check how many parking spaces are available during any giving day, and check the parking price for there stay at the car par, this API also allows the user to see the price during holiday times and weekends as these days are more expensive. The user can check their booking information via their car registration number and cancel their booking and/or amend their booking information.

C#, .NET, Entity Framework

To see the source code please check out my GitHub

Dissertation Project

Virtual reality

Language Used: Unity3D, C#

Retired Projects

Year 2 Project

Repulsive Dreams

This small demo was created in my second year at university. The player needs to collect keys to progress through the mansion, however many monsters are lurking within the shadows of the building. There are many different features built into the demo, but I'm most proud of the inventory system that I built. It was based on the world of warcraft inventory system. All game mechanics were built by me along with adding in the animations for the player.
