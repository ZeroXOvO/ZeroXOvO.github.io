Repulsive Dreams

A Horror adventure game

This was a second-year project I did. The game is a third person horror game inspired by the resident evil series. The game contains a functional HUD containing both a health and stamina bar, the stamina bar is connected to the run function and the stamina regains once either the player stops running, or the bar runs out, making the player move slow until the bar is refilled. The game also has a complexed inventory system, where items gathered throughout the level are stored and can be accesses whenever the player needed them.

Language Used: Unreal Engine, Blueprints
